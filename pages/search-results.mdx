# Search Results
this page has a lots of methouds. most of them are passed to `<FilterAndSortBar />` to let user make actions in search form.
## `toggleShowFilters()`
this will set `showFilters` state value. 
## `toggleDeliverAtRentersPlace`
this will set `DeliverAtRentersPlace` state value and then call `this.renderResults()` again to refetch search results data.
